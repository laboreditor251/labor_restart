# 本地图片目录说明

- 建议把故事配图放在本目录：`assets/images/`
- 在 `assets/story.json` 的节点中使用本地路径，例如：
  - `\"image\": \"/assets/images/b1_start.jpg\"`
  - `\"image\": \"./assets/images/b4_a.png\"`
- 推荐命名方式：`节点ID.扩展名`
  - 例：`b1_start.jpg`、`b4_a.webp`
- 建议尺寸：宽图 `1600x900` 或接近比例，文件尽量压缩到 `500KB` 内

当前示例：`b1_start` 已配置 `\"image\": \"/assets/images/b1_start.jpg\"`。
